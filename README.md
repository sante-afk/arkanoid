# Arkanoid

Classic Arkanoid (breakout) game — test assignment for Studio Perets

## Description

A browser-based implementation of the classic Arkanoid game, developed as a technical test for Studio Perets. The player controls a paddle to bounce a ball and break bricks, with the goal of clearing all bricks from the screen.

## Features

- Classic paddle-and-ball mechanics
- Brick-breaking gameplay
- Smooth controls
- Responsive design

## Technologies

- JavaScript (81.1%)
- CSS (13.2%)
- HTML (5.7%)
